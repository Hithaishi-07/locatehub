# LocateHub - Nearby Services Finder

A beautiful and fully functional **location-based service finder** web app.

## ✨ Features
- Interactive Map using Leaflet
- Live Geolocation ("My Location" button)
- Search & Category Filters (Food, Hospital, Transport, Shopping)
- Responsive Design (Mobile + Desktop)
- Smooth animations

## 🚀 Live Demo
[View Live App](https://yourusername.github.io/locatehub)

## 🛠️ Technologies Used
- HTML5
- Tailwind CSS
- Leaflet.js (OpenStreetMap)
- Vanilla JavaScript

## How to Run Locally
1. Download `index.html`
2. Open it in any web browser (Chrome/Firefox/Edge)
